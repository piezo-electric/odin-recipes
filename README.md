# odin-recipes website project

## About
This is my first project detailing my knowledge on HTML boilerplate formation and basic website design for The Odin Project (TOP). 

Also, this project serves as a git practice session for learning proper git commit etiquette. 

My key objectives for this lesson are to understand HTML tags and correctly apply them in the formation of a basic recipe website. I show examples of:

1. Applying relative and absolute links.
2. The use of proper syntax for images, including gifs and jpgs with <img> tag.
3. Creating unordered and ordered lists.

With that, that I hope you enjoy visiting my website and looking at my code for idea. :)
